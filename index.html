Animations

Today, modern technologies allow us to animate static web pages and applications using program code, so we can get rid of heavy video and GIF formats. Now our animations look more beautiful and uncompressed at any scale. For myself, I choose the wonderful library - Lottie. My whole process of creating animation is divided into the following steps: I make vector illustration, import it into Adobe After Effects and work on my animation here, then I finish it, export the animation code and use it on a website or mobile application.
